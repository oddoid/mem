# mem-font
4x4 fixed width pixel font

## Installing
 npm i -S mem-font

## Build Dependencies
- fontforge
- imagemagick
- potrace

Initial Gimp guidelines were drawn using the
[Grid of Guides script](http://registry.gimp.org/node/12003)
